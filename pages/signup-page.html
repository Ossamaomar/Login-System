A valid Username:
  • Must start with a letter.
  • Can contain letters, numbers, underscores, or periods.
  • Must be 3-16 characters long.
A valid email address should:
  • Start with letters, digits, or special characters like ., _, %, +, or -.
  • Include a @ symbol followed by a valid domain (e.g., example.com).
  • Have a domain that ends with a valid top-level domain (e.g., .com, .net).
A valid password should:
  • Be at least 8 characters long.
  • Include at least one uppercase letter (A-Z) and at least one lowercase letter (a-z).
  • Include at least one number (0-9).
  • Include at least one special character (e.g., @, $, !, %, *, ?, or &).
